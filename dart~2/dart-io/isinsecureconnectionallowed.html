Whether insecure connections to host are allowed.

This API is deprecated and always returns true. See https://github.com/flutter/flutter/issues/72723 for more details.

host must be a String or InternetAddress.

If any of the domain policies match host, the matching policy will make the decision. If multiple policies apply, the top matching policy makes the decision. If none of the domain policies match, the embedder default is used.

Loopback addresses are always allowed.
