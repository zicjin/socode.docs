T squaredDistanceTo(
  1. Point<T> other

Returns the squared distance between this and other.

Squared distances can be used for comparisons when the actual value is not required.

Implementation

T squaredDistanceTo(Point<T> other) {
  var dx = x - other.x;
  var dy = y - other.y;
  return (dx * dx + dy * dy) as T;
}
