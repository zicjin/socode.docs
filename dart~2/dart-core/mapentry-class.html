MapEntry<K, V> class

A key/value pair representing an entry in a Map.

The Map interface contains various methods that can inspect or modify the map based on entry objects.

final map = {'1': 'A', '2': 'B'};
map.addEntries([
 MapEntry('3', 'C'),
 MapEntry('4', 'D'),
]);
print(map); // {1: A, 2: B, 3: C, 4: D}
