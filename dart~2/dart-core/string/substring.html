substring method

String substring(
  1. int start,
  2. [int? end]
)

The substring of this string from start,inclusive, to end, exclusive.

Example:

var string = 'dartlang';
string.substring(1);    // 'artlang'
string.substring(1, 4); // 'art'

Both start and end must be non-negative and no greater than length, and end, if provided, must be greater than or equal to start.
