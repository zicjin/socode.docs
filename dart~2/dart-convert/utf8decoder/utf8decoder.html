Instantiates a new Utf8Decoder.

The optional allowMalformed argument defines how convert deals with invalid or unterminated character sequences.

If it is true, convert replaces invalid (or unterminated) character sequences with the Unicode Replacement character U+FFFD (ï¿½). Otherwise it throws a FormatException.
