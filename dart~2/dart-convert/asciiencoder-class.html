dart:convert

AsciiEncoder class

Converts strings of only ASCII characters to bytes.

Example:

const asciiEncoder = AsciiEncoder();
const sample = 'Dart';
final asciiValues = asciiEncoder.convert(sample);
print(asciiValues); // [68, 97, 114, 116]
