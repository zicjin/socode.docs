indexedDB property

IdbFactory? indexedDB
@SupportedBrowser(SupportedBrowser.CHROME, '23.0'), @SupportedBrowser(SupportedBrowser.FIREFOX, '15.0'), @SupportedBrowser(SupportedBrowser.IE, '10.0')

Gets an instance of the Indexed DB factory to being using Indexed DB.

Use indexed_db.IdbFactory.supported to check if Indexed DB is supported on the current platform.
