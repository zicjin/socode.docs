kotlin-stdlib / kotlin.native.concurrent / AtomicReference / value
Platform and version requirements: Native (1.3) 
var value: T

The referenced value. Gets the value or sets the new value. If new value is not null, it must be frozen or permanent object.

Exceptions

InvalidMutabilityException - if the value is not frozen or a permanent object
