Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
fun toULong(): ULong

Converts this UInt value to ULong.

The resulting ULong value represents the same numerical value as this UInt.

The least significant 32 bits of the resulting ULong value are the same as the bits of this UInt value, whereas the most significant 32 bits are filled with zeros.
