fromClosedRange

kotlin-stdlib / kotlin.ranges / LongProgression / fromClosedRange
Platform and version requirements: JVM (1.0), JS (1.1), Native (1.3) 
fun fromClosedRange(
    rangeStart: Long, 
    rangeEnd: Long, 
    step: Long
): LongProgression

Creates LongProgression within the specified bounds of a closed range.

The progression starts with the rangeStart value and goes toward the rangeEnd value not excluding it, with the specified step. In order to go backwards the step must be negative.

step must be greater than Long.MIN_VALUE and not equal to zero.
