Supported Azure resources in Terraform Cloud Cost Estimation

Note: Cost estimation is a paid feature, available as part of the Team & Governance upgrade package. Learn more about Terraform Cloud pricing here.

Terraform Cloud can estimate monthly costs for many Azure Terraform resources.

Note: Terraform Enterprise requires Azure credentials to support cost estimation. These credentials are configured at the instance level, not the organization level. See the Application Administration docs for more details.

Handled Resources

Below is a list of resources that cost estimation supports:

  • azurerm_app_service_environment
  • azurerm_app_service_plan
