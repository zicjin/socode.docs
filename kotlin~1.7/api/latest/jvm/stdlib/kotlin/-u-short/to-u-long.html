fun toULong(): ULong

Converts this UShort value to ULong.

The resulting ULong value represents the same numerical value as this UShort.

The least significant 16 bits of the resulting ULong value are the same as the bits of this UShort value, whereas the most significant 48 bits are filled with zeros.
