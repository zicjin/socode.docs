fun toShort(): Short

Converts this ULong value to Short.

If this value is less than or equals to Short.MAX_VALUE, the resulting Short value represents the same numerical value as this ULong.

The resulting Short value is represented by the least significant 16 bits of this ULong value. Note that the resulting Short value may be negative.
