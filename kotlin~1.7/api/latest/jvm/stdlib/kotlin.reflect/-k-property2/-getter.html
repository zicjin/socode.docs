interface Getter<D, E, out V> : 
    KProperty.Getter<V>, 
    (D, E) -> V

Getter of the property is a get method declared alongside the property.

Can be used as a function that takes an argument of type D (the first receiver), an argument of type E (the second receiver) and returns the value of the property type V.
