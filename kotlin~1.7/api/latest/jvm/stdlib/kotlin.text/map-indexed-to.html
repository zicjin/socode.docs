mapIndexedTo

kotlin-stdlib / kotlin.text / mapIndexedTo
Platform and version requirements: JVM (1.0), JS (1.0), Native (1.0) 
inline fun <R, C : MutableCollection<in R>> CharSequence.mapIndexedTo(
    destination: C, 
    transform: (index: Int, Char) -> R
): C

Applies the given transform function to each character and its index in the original char sequence and appends the results to the given destination.

Parameters

transform - function that takes the index of a character and the character itself and returns the result of the transform applied to the character.
