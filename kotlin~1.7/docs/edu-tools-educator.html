Teaching Kotlin with EduTools plugin

With the EduTools plugin, available both in Android Studio and IntelliJ IDEA, you can teach Kotlin through code practicing tasks. Take a look at the Educator Start Guide to learn how to create a simple Kotlin course that includes a set of programming tasks and integrated tests.

If you want to use the EduTools plugin to learn Kotlin, read Learning Kotlin with EduTools plugin.
