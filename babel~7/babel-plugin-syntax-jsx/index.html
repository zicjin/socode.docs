@babel/plugin-syntax-jsx

Syntax only

Using this plugin directly only enables Babel to parse this syntax. If you want to remove JSX syntax then use the transform-react-jsx plugin or react preset to both parse and transform this syntax.

Installation

npm install --save-dev @babel/plugin-syntax-jsx

Usage

With a configuration file (Recommended)

{
  "plugins": ["@babel/plugin-syntax-jsx"]
}

Via CLI

babel --plugins @babel/plugin-syntax-jsx script.js

Via Node API

require("@babel/core").transformSync("code", {
  plugins: ["@babel/plugin-syntax-jsx"],
});
