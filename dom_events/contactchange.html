contactchange

The contactchange event is fired each time a contact is added, updated or removed from the device's contact database.

Bubbles No
Cancelable No
Target objects ContactManager
Interface MozContactChangeEvent

Properties

Property Type Description
target Read only EventTarget The event target (the topmost target in the DOM tree).
type Read only DOMString The type of event.
bubbles Read only boolean Does the event normally bubble?
cancelable Read only boolean Is it possible to cancel the event?
contactID Read only DOMString The unique ID of the contact that has been changed.
reason Read only DOMString The reason why the contact has been updated.
