submit

The submit event is fired when a form is submitted.

Note that submit is fired only on the form element, not the button or submit input. (Forms are submitted, not buttons.)

Trying to submit a form that does not pass validation triggers an invalid event instead. The validation prevents submitting the form, thus there is no submit event.
