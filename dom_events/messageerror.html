messageerror (Web Workers)

The messageerror event (used by multiple APIs: MessagePort, Web Workers, Broadcast Channel, Window) is fired when a object receives a message that cannot be deserialized and therefore raises an error.

Bubbles No
Cancelable No
Target objects MessagePort, DedicatedWorkerGlobalScope, Worker, BroadcastChannel, Window
Interface MessageEvent

Examples

Worker.onmessageerror = function() {
  // Respond to a message error
};

Inheritance

The messageerror event implements the MessageEvent interface, which inherits from Event. It therefore has available the properties and methods defined on both these interfaces.
