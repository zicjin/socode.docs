connect (Web Workers)

The connect event is fired in shared workers at their SharedWorkerGlobalScope when a new client connects.

General info

Specification
HTML5
Interface
MessageEvent
Bubbles
No
Cancelable
No
Target
SharedWorkerGlobalScope
Default Action
None

Properties

Property Type Description
source Read only MessagePort The port of the newly connected client.
