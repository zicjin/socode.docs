promise public

Module: @ember/object

Defined in packages/@ember/-internals/runtime/lib/mixins/promise_proxy.js:151

The promise whose fulfillment value is being proxied by this object.

This property must be specified upon creation, and should not be changed once created.

Example:

import ObjectProxy from '@ember/object/proxy';
import PromiseProxyMixin from '@ember/object/promise-proxy-mixin';

ObjectProxy.extend(PromiseProxyMixin).create({
  promise: <thenable>
});
