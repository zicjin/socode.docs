Returns true if the passed value is null or undefined. This avoids errors from JSLint complaining about use of ==, which can be technically confusing.

isNone();              // true
isNone(null);          // true
isNone(undefined);     // true
isNone('');            // false
isNone([]);            // false
isNone(function() {}); // false
