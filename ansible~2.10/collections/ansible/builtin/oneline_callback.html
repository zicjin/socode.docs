ansible.builtin.oneline – oneline Ansible screen output

Note

This module is part of ansible-base and included in all Ansible installations. In most cases, you can use the short module name oneline even without specifying the collections: keyword. Despite that, we recommend you use the FQCN for easy linking to the module documentation and to avoid conflicting with other collections that may have the same module name.

Synopsis

  • This is the output callback used by the -o/–one-line command line option.
