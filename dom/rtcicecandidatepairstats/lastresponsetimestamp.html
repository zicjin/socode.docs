RTCIceCandidateStats.lastResponseTimestamp

The RTCIceCandidatePairStats property lastResponseTimestamp indicates the time at which the last STUN response was received on the described candidate pair.

Syntax

lastResponseTimestamp = rtcIceCandidatePairStats.lastResponseTimestamp;

Value

A DOMHighResTimeStamp object indicating the timestamp at which the most recent STUN response was received on the connection defined by the described pair of candidates.
