XMLHttpRequest.msCachingEnabled()

Non-standard: This feature is non-standard and is not on a standards track. Do not use it on production sites facing the Web: it will not work for every user. There may also be large incompatibilities between implementations and the behavior may change in the future.

The msCachingEnabled() method gets the current caching state for an XMLHttpRequest.

Syntax

msCachingEnabled()

Parameters

None.

Return value

A boolean value. If true, the XMLHttpRequest is cached to disk. If false, it is not written to disk.

Specifications

This feature is not part of any current specification. It is no longer on track to become a standard.
