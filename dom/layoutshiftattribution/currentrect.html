LayoutShiftAttribution.currentRect

The currentRect read-only property of the LayoutShiftAttribution interface returns a DOMRectReadOnly object representing the position of the element before the shift.

Syntax

let currentRect = LayoutShiftAttribution.currentRect;

Value

A DOMRectReadOnly object.

Examples

The following example prints the currentRect of the first item in LayoutShift.sources to the console.

new PerformanceObserver((list) => {
  for (const {sources} of list.getEntries()) {
    if (sources) {
      console.log(sources[0].currentRect);
    }
  }
}).observe({type: 'layout-shift', buffered: true});
