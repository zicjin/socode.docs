RTCStats.type

The RTCStats dictionary's property type is a string which specifies the type of statistic represented by the object, where the permitted values are drawn from the enum type RTCStatsType. The string can be used to determine which of the RTCStats-based dictionaries are the foundation of the statistics object.

Syntax

var type = RTCStats.type;

Value

A DOMString which specifies which type of statistic is represented by the object. The string comes from the RTCStatsType enum and corrsponds to one of the RTCStats-based statistic object types.
