MIDIOutput.clear()

Secure context

This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The clear() method of the MIDIOutput interface clears the queue of messages being sent to the output device.

Syntax

MIDIOutput.clear();

Parameters

None.
