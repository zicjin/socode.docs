XRSession: inputsourceschange event

The inputsourceschange event is sent to an XRSession when the set of available WebXR input devices changes.

The received event, of type XRInputSourcesChangeEvent, contains a list of any newly added and/or removed input devices.

Bubbles Yes
Cancelable No
Interface XRInputSourcesChangeEvent
Event handler property oninputsourceschange

The event object contains lists of the newly-added and/or removed input devices in its added and removed properties.

See XRInputSourcesChangeEvent for example code.
