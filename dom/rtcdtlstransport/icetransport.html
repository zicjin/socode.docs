RTCDtlsTransport.iceTransport

The read-only RTCDtlsTransport property iceTransport contains areference to the underlying RTCIceTransport.

Syntax

var iceTransport = rtcDtlsTransport.iceTransport;

Value

The underlying RTCIceTransport instance.
