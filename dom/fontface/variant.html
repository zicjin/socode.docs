FontFace.variant

The variant property of the FontFace interface programmatically retrieves or sets font variant values. It is equivalent to the font-variant descriptor.

Syntax

var variantSubProperty = FontFace.variant;
FontFace.variant = variantSubProperty;

Value

A CSSOMString containing a descriptor as it would be defined in a style sheet's @font-face rule.

Specifications
