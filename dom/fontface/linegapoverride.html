FontFace.lineGapOverride

The lineGapOverride property of the FontFace interface returns and sets the value of the @font-face/line-gap-override descriptor. The possible values are normal, indicating that the metric used should be obtained from the font file, or a percentage.

Syntax

let lineGapOverride = FontFace.lineGapOverride;
FontFace.lineGapOverride = '90%';

Value

A string.

Examples

let fontFace = new FontFace('Roboto', 'url(https://fonts.example.com/roboto.woff2)', {'lineGapOverride':'90%'});
console.log(fontFace.lineGapOverride); // 90%
fontFace.lineGapOverride = 'normal';
console.log(fontFace.lineGapOverride); // 'normal'
