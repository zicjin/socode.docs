FontFace.variationSettings

The variationSettings property of the FontFace interface retrieves or sets low-level OpenType or TrueType font variations. It is equivalent to the font-variation-settings descriptor.

Syntax

let variationSettingDescriptor = FontFace.variationSettings;
FontFace.variationSettings = variationSettingDescriptor;

Value

A CSSOMString containing a descriptor.

Specifications
