HTMLTableElement.tFoot

The HTMLTableElement.tFoot property represents the <tfoot> element of a <table>. Its value will be null if there is no such element.

Value

A <tfoot> element or null.

Examples

if (table.tFoot === my_foot) {
  // …
}
