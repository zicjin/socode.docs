BackgroundFetchRegistration.id

The id read-only property of the BackgroundFetchRegistration interface returns a copy of the background fetch's ID.

Syntax

let id = BackgroundFetchRegistration.id;

Value

A string.

Examples

Logging the BackgroundFetchRegistration.id to the console returns the ID set when registering the fetch. In this case, "my-fetch".

console.log(bgFetch.id); // "my-fetch"
