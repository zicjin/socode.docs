RTCInboundRtpStreamStats.remoteId

The remoteId property of the RTCInboundRtpStreamStats dictionary specifies the id of the RTCRemoteOutboundRtpStreamStats object representing the remote peer's RTCRtpSender which is sending the media to the local peer.

Syntax

var remoteStatsId = rtcInboundRtpStreamStats.remoteId;

Value

A DOMString containing the ID of the RTCRemoteOutboundRtpStreamStats object that represents the remote peer's RTCRtpSender for the synchronization source represented by this stats object.
