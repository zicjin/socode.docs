SharedWorker.port

The port property of the SharedWorker interface returns a MessagePort object used to communicate and control the shared worker.

Syntax

myWorker.port;

Value

A MessagePort object.

Example

The following code snippet shows creation of a SharedWorker object using the SharedWorker() constructor. Multiple scripts can then access the worker through a MessagePort object accessed using the SharedWorker.port property â€” the port is started using its start() method:

var myWorker = new SharedWorker('worker.js');
myWorker.port.start();

For a full example, see our Basic shared worker example (run shared worker.)
