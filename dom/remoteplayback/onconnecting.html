RemotePlayback.onconnecting

The onconnecting event handler of the RemotePlayback interface processes the events fired when the user agent initiates remote playback.

Syntax

RemotePlayback.onconnecting = function;
RemotePlayback.addEventListener('connecting', function);

Example

In the following example the value of RemotePlayback.state is printed to the console when the user agent initiates a connection.

RemotePlayback.onconnecting = function() {
  console.log(RemotePlayback.state);
}
