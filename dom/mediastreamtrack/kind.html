MediaStreamTrack.kind

The MediaStreamTrack.kind read-only property returns a string set to "audio" if the track is an audio track and to "video", if it is a video track. It doesn't change if the track is disassociated from its source.

Value

The possible values are a string with one of the following values:

  • "audio": the track is an audio track.
  • "video": the track is a video track.
