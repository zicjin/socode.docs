MIDIInput.onmidimessage

Secure context

This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The onmidimessage event handler of the MIDIInput interface processes midimessage events.

The event fires when the MIDI port corresponding to this MIDIInput finishes receiving one or more MIDI messages. An instance of MIDIMessageEvent containing the message that was received is passed to this event handler.

Syntax

MIDIInput.onmidimessage = function;
MIDIInput.addEventListener('midimessage', function);

Examples

In the following example onmidimessage events are listened for on all input ports. When a message is received the MIDIMessageEvent.data property is printed to the console.

inputs.forEach((input) => {
  input.onmidimessage = function(message) {
    console.log(message.data);
  }
})
