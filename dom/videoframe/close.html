VideoFrame.close()

The close() method of the VideoFrame interface clears all states and releases the reference to the media resource.

Syntax

VideoFrame.close()

Parameters

None.

Return Value

Undefined.

Examples

The following example shows the VideoFrame object being closed.

VideoFrame.close();
