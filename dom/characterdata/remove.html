CharacterData.remove()

The CharacterData.remove() method removes text.

Syntax

remove()

Examples

Using remove() 

<p id="myText">Some text</p>

let text = document.getElementById('myText').firstChild;
text.remove(); // Removes the text

<p id="myText"></p>
