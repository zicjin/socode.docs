Sensor.timestamp

The timestamp read-only property of the Sensor interface returns the time stamp of the latest sensor reading.

Syntax

var timestamp = sensorInstance.timestamp

Because Sensor is a base class, timestamp may only be read from one of its derived classes.

Value

A DOMHighResTimeStamp.
