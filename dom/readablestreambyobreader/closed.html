The closed read-only property of the ReadableStreamBYOBReader interface returns a Promise that fulfills when the stream closes or the reader's lock is released, or rejects if the stream throws an error. This property enables you to write code that responds to an end to the streaming process.

Syntax

var closed = readableStreamBYOBReader.closed;

Value

A Promise.
