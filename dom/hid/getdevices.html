HID.getDevices()

Secure context

This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The getDevices() method of the HID interface gets a list of the connected HID devices.

Syntax

HID.getDevices();

Parameters

None.

Return value

A Promise that resolves with a list of HIDDevice objects.

Examples

The following example gets a list of devices and logs the device names to the console.

document.addEventListener('DOMContentLoaded', async () => {
  let devices = await navigator.hid.getDevices();
  devices.forEach(device => {
    console.log(`HID: ${device.productName}`);
  });
});
