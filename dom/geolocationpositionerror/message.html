GeolocationPositionError.message

Secure context

This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The GeolocationPositionError.message read-only property returns a human-readable DOMString describing the details of the error.

Syntax

let msg = geolocationPositionErrorInstance.message

Value

A human-readable DOMString describing the details of the error.
