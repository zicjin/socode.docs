GeolocationPosition.timestamp

Secure context: This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The GeolocationPosition.timestamp read-only property returns a EpochTimeStamp object that represents the date and time that the position was acquired by the device.

Value

An EpochTimeStamp object instance indicating the time that the position was acquired.
