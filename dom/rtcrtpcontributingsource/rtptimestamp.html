RTCRtpContributingSource.rtpTimestamp

The read-only rtpTimestamp property of the RTCRtpContributingSource dictionary contains a DOMHighResTimeStamp indicating the source-generated time at which the media contained int he packet was first sampled or obtained.

Value

An integer value specifying a source-generated timestamp indicating the time at which the media in this packet, scheduled for play out at the time indicated by timestamp, was initially sampled or generated. This value may be useful for sequencing and synchronization purposes.
