ElementInternals.role

The role read-only property of the ElementInternals interface returns the WAI-ARIA role for the element. For example, a checkbox might have role="checkbox

Syntax

let role = ElementInternals.role;

Value

A DOMString which contains an ARIA role. A full list of ARIA roles can be found on the ARIA techniques page.
