Notification.actions

Note: This feature is available in Web Workers

Secure context: This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The actions read-only property of the Notification interface provides the actions available for users to choose from for interacting with the notification.

The actions are set using the actions option of the second argument for the showNotification() method and Notification() constructor.

Note: Browsers typically limit the maximum number of actions they will display for a particular notification. Check the static Notification.maxActions property to determine the limit.

Value

A read-only array of actions. Each element in the array is an object with the following members:

action

A string identifying a user action to be displayed on the notification.

title

A string containing action text to be shown to the user.

icon

A string containing the URL of an icon to display with the action.
