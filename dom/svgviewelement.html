SVGViewElement

The SVGViewElement interface provides access to the properties of <view> elements, as well as methods to manipulate them.

Properties

This interface also inherits properties from its parent interface, SVGElement.

SVGViewElement.viewTarget Deprecated

An SVGStringList corresponding to the viewTarget attribute of the given <view> element. A list of string values which contain the names listed in the viewTarget attribute. Each of the string values can be associated with the corresponding element using the getElementById() method call.

Methods

This interface doesn't implement any specific methods, but inherits methods from its parent interface, SVGElement.
