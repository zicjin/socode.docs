SecurityPolicyViolationEvent.effectiveDirective

The effectiveDirective read-only property of the SecurityPolicyViolationEvent interface is a string representing the directive whose enforcement uncovered the violation.

Value

A string representing the directive whose enforcement uncovered the violation.

Examples

document.addEventListener("securitypolicyviolation", (e) => {
  console.log(e.effectiveDirective);
});
