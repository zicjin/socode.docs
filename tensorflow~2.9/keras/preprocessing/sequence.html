Module: tf.keras.preprocessing.sequence

Utilities for preprocessing sequence data.

Deprecated: tf.keras.preprocessing.sequence APIs are not recommended for new code. Prefer tf.keras.utils.timeseries_dataset_from_array and the tf.data APIs which provide a much more flexible mechanisms for dealing with sequences. See the tf.data guide for more details.

Classes

class TimeseriesGenerator: Utility class for generating batches of temporal data.

Functions

make_sampling_table(...): Generates a word rank-based probabilistic sampling table.

pad_sequences(...): Pads sequences to the same length.

skipgrams(...): Generates skipgram word pairs.
