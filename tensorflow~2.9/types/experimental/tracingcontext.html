tf.types.experimental.TracingContext

Contains information scoped to the tracing of multiple objects.

TracingContext is a container class for flags and variables that have any kind of influence on the tracing behaviour of the class implementing the tf_tracing_type. This context will be shared across all tf_tracing_type calls while constructing the TraceType for a particular set of objects.
