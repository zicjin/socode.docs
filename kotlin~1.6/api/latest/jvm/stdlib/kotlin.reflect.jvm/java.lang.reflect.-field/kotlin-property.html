kotlinProperty

kotlin-stdlib / kotlin.reflect.jvm / java.lang.reflect.Field / kotlinProperty
Platform and version requirements: JVM (1.0) 
val Field.kotlinProperty: KProperty<*>?

Returns a KProperty instance corresponding to the given Java Field instance, or null if this field cannot be represented by a Kotlin property (for example, if it is a synthetic field).
