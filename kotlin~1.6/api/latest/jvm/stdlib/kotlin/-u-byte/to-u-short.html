fun toUShort(): UShort

Converts this UByte value to UShort.

The resulting UShort value represents the same numerical value as this UByte.

The least significant 8 bits of the resulting UShort value are the same as the bits of this UByte value, whereas the most significant 8 bits are filled with zeros.
