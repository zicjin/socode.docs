fun toInt(): Int

Converts this ULong value to Int.

If this value is less than or equals to Int.MAX_VALUE, the resulting Int value represents the same numerical value as this ULong.

The resulting Int value is represented by the least significant 32 bits of this ULong value. Note that the resulting Int value may be negative.
