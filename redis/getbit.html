GETBIT

GETBIT key offset

Available since 2.2.0.

Time complexity: O(1)

Returns the bit value at offset in the string value stored at key.

When offset is beyond the string length, the string is assumed to be a contiguous space with 0 bits. When key does not exist it is assumed to be an empty string, so offset is always out of range and the value is also assumed to be a contiguous space with 0 bits.

Return value

Integer reply: the bit value stored at offset.

Examples

redis> SETBIT mykey 7 1 ERR Unknown or disabled command 'SETBIT' redis> GETBIT mykey 0 ERR Unknown or disabled command 'GETBIT' redis> GETBIT mykey 7 ERR Unknown or disabled command 'GETBIT' redis> GETBIT mykey 100 ERR Unknown or disabled command 'GETBIT'
