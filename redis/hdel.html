HDEL

HDEL key field [field ...]

Available since 2.0.0.

Time complexity: O(N) where N is the number of fields to be removed.

Removes the specified fields from the hash stored at key. Specified fields that do not exist within this hash are ignored. If key does not exist, it is treated as an empty hash and this command returns 0.

Return value

Integer reply: the number of fields that were removed from the hash, not including specified but non existing fields.

Examples

redis> HSET myhash field1 "foo" (integer) 1 redis> HDEL myhash field1 (integer) 1 redis> HDEL myhash field2 (integer) 0

History

  • Redis version >= 2.4.0: Accepts multiple `field` arguments.
