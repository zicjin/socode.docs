C++ keywords: and_eq

Usage

  • alternative operators: as an alternative for &=

Example

#include <iostream>
#include <bitset>
int main()
{
    std::bitset<4> mask("1100");
    std::bitset<4> val("0111");
    val and_eq mask;
    std::cout << val << '\n';
}

Output:

0100
