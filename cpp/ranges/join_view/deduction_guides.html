deduction guides for std::ranges::join_view 

template<class R>
explicit join_view(R&&) -> join_view<views::all_t<R>>;
 (since C++20)

The deduction guide is provided for std::ranges::join_view to allow deduction from range.

If R is a view, the deduced type is join_view<R>; otherwise, the deduced type is usually join_view<ranges::ref_view<R>>.
