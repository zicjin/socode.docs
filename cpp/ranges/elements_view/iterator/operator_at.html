std::ranges::elements_view<V,F>::iterator<Const>::operator[]

constexpr decltype(auto) operator[]( difference_type n ) const
  requires ranges::random_access_range<Base>;
 (since C++20)

Returns the element at specified relative location.

Effectively returns /*get-element*/(this->base() + n), where for an expression e, /*get-element*/(e) is.

  • std::get<N>(*e), if ranges::range_reference_t<Base> is a reference type,
  • otherwise, static_cast<E>(std::get<N>(*e)), where E is std::remove_cv_t<std::tuple_element_t<N, ranges::range_reference_t<Base>>>.

Parameters

n - position relative to current location.

Return value

the element at displacement n relative to the current location.
