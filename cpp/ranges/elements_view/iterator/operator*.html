std::ranges::elements_view<V,F>::iterator<Const>::operator*

constexpr decltype(auto) operator*() const;
 (since C++20)

Returns the element into V the underlying iterator points to.

Effectively returns /*get-element*/(this->base()), where for an expression e, /*get-element*/(e) is.

  • std::get<N>(*e), if ranges::range_reference_t<Base> is a reference type,
  • otherwise, static_cast<E>(std::get<N>(*e)), where E is std::remove_cv_t<std::tuple_element_t<N, ranges::range_reference_t<Base>>>.

Parameters

(none).

Return value

The current element.

Notes

operator-> is not provided.
