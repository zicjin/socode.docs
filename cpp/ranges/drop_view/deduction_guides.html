deduction guides for std::ranges::drop_view

Defined in header <ranges> 
template< class R >
drop_view( R&&, ranges::range_difference_t<R> ) -> drop_view<views::all_t<R>>;
 (since C++20)

The deduction guide is provided for std::ranges::drop_view to allow deduction from range and number of elements.

If R is a view, the deduced type is drop_view<R>; otherwise, the deduced type is usually drop_view<ranges::ref_view<R>>.
