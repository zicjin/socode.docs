Specializes the std::swap algorithm for std::basic_string. Swaps the contents of lhs and rhs. Equivalent to lhs.swap(rhs).

Parameters

lhs, rhs - strings whose contents to swap

Return value

(none).

Complexity

Constant.

Exceptions

noexcept specification:
noexcept(noexcept(lhs.swap(rhs)))
 (since C++17)

Example

#include <string>
#include <iostream>
 
int main() 
{
    std::string a = "AAAA";
    std::string b = "BBB";
 
    std::cout << "before swap" << '\n';
    std::cout << "a: " << a << '\n';
    std::cout << "b: " << b << '\n';
 
    std::swap(a,b);
 
    std::cout << "after swap" << '\n';
    std::cout << "a: " << a << '\n';
    std::cout << "b: " << b << '\n';
}

Output:

before swap
a: AAAA
b: BBB
after swap
a: BBB
b: AAAA
