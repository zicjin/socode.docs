pandas.Series.dt

Series.dt() [source]

Accessor object for datetimelike properties of the Series values.

Examples

>>> s.dt.hour
>>> s.dt.second
>>> s.dt.quarter

Returns a Series indexed like the original Series. Raises TypeError if the Series does not contain datetimelike values.
