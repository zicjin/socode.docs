Information Schema INNODB_SYS_DATAFILES Table

MariaDB starting with 10.0

The INNODB_SYS_DATAFILES table was added in MariaDB 10.0.4.

The Information Schema INNODB_SYS_DATAFILES table contains information about InnoDB datafile paths. The PROCESS privilege is required to view the table.

It contains the following columns:

Column Description
SPACE Numeric tablespace. Matches the INNODB_SYS_TABLES.SPACE value.
PATH Tablespace datafile path.

Example

SELECT * FROM INNODB_SYS_DATAFILES;
+-------+--------------------------------+
| SPACE | PATH                           |
+-------+--------------------------------+
|    19 | ./test/t2.ibd                  |
|    20 | ./test/t3.ibd                  |
...
|    68 | ./test/animals.ibd             |
|    69 | ./test/animal_count.ibd        |
|    70 | ./test/t.ibd                   |
+-------+--------------------------------+
