JSON_DETAILED

MariaDB starting with 10.2.4

This function was added in MariaDB 10.2.4.

Syntax

JSON_DETAILED(json_doc[, tab_size])

Description

Represents JSON in the most understandable way emphasizing nested structures.

Example

SET @j = '{ "A":1,"B":[2,3]}';

SELECT @j;
+--------------------+
| @j                 |
+--------------------+
| { "A":1,"B":[2,3]} |
+--------------------+

SELECT JSON_DETAILED(@j);
+------------------------------------------------------------+
| JSON_DETAILED(@j)                                          |
+------------------------------------------------------------+
| {
    "A": 1,
    "B": 
    [
        2,
        3
    ]
} |
+------------------------------------------------------------+
