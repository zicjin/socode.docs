OCTET_LENGTH

Syntax

OCTET_LENGTH(str)

Description

OCTET_LENGTH() is normally a synonym for LENGTH(). When running Oracle mode from MariaDB 10.3, they are not synonyms, but OCTET_LENGTH() behaves as LENGTH() would when not in Oracle mode.
