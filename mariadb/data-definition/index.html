Data Definition

SQL Commands for defining data, such as ALTER, CREATE, DROP, RENAME etc.

Title Description
ALTER This category is for documentation on the various ALTER statements.
CONSTRAINT Define a CHECK or FOREIGN KEY constraint.
CREATE Articles on the various CREATE statements.
DROP Articles on various DROP commands.
MERGE Allows you to access a collection of identical MyISAM tables as one.
RENAME TABLE Change a table's name.
TRUNCATE TABLE DROP and re-CREATE a table.
