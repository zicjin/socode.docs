SHOW QUERY_RESPONSE_TIME

MariaDB starting with 10.1.1

SHOW QUERY_RESPONSE_TIME was introduced in MariaDB 10.1.1.

Starting with MariaDB 10.1.1, which introduced the Information Schema plugin extension, it is possible to use SHOW QUERY_RESPONSE_TIME as an alternative for retrieving information from the QUERY_RESPONSE_TIME plugin.
