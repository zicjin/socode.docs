SEQUENCE Functions

MariaDB starting with 10.3

SEQUENCEs were introduced in MariaDB 10.3.

Functions that can be used on SEQUENCEs

Title Description
LASTVAL Synonym for PREVIOUS VALUE for sequence_name.
NEXT VALUE for sequence_name Generate next value for a SEQUENCE. Same as NEXTVAL().
NEXTVAL Synonym for NEXT VALUE for sequence_name.
PREVIOUS VALUE FOR sequence_name Get last value generated from a SEQUENCE. Same as LASTVAL().
SETVAL() Set the next value to be returned from a SEQUENCE.
