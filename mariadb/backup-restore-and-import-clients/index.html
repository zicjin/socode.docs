Backup, Restore and Import Clients

Clients for taking backups or importing/restoring data

Title Description
Mariabackup Physical backups, supports Data-at-Rest and InnoDB compression.
Percona XtraBackup Open source tool for performing hot backups of MariaDB, MySQL and Percona Server databases.
mariadb-dump Symlink or new name for mysqldump.
mariadb-hotcopy Symlink or new name for mysqlhotcopy.
mariadb-import Symlink or new name for mysqlimport.
mysqldump Dump a database or a collection of databases in a portable format.
mysqlhotcopy Fast backup program on local machine. Deprecated.
mysqlimport Loads tables from text files in various formats.
