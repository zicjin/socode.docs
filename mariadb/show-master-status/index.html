SHOW MASTER STATUS

Syntax

SHOW MASTER STATUS

Description

Provides status information about the binary log files of the master.

This statement requires the SUPER or the REPLICATION_CLIENT privilege.

To see information about the current GTIDs in the binary log, use the gtid_binlog_pos variable.

Example

SHOW MASTER STATUS;
+--------------------+----------+--------------+------------------+
| File               | Position | Binlog_Do_DB | Binlog_Ignore_DB |
+--------------------+----------+--------------+------------------+
| mariadb-bin.000016 |      475 |              |                  |
+--------------------+----------+--------------+------------------+
SELECT @@global.gtid_binlog_pos;
+--------------------------+
| @@global.gtid_binlog_pos |
+--------------------------+
| 0-1-2                    |
+--------------------------+
