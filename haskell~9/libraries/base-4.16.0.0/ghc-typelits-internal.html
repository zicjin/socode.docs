GHC.TypeLits.Internal

Safe Haskell Trustworthy
Language Haskell2010

Description

This module exports the Type Literal kinds as well as the comparison type families for those kinds. It is needed to prevent module cycles while still allowing these identifiers to be improted in Ord.

Since: base-4.16.0.0
