type family OrdCond o lt eq gt where ... Source

A case statement on Ordering. `Ordering c l e g` is l when `c ~ LT`, e when `c ~ EQ`, and g when `c ~ GT`. @since 4.16.0.0

Equations

OrdCond 'LT lt eq gt = lt
OrdCond 'EQ lt eq gt = eq
OrdCond 'GT lt eq gt = gt
