Stores a value if the current value is the same as the old value.

The stabilized version of this intrinsic is available on the atomic types via the compare_exchange method by passing Ordering::AcqRel as the success and Ordering::Acquire as the failure parameters. For example, AtomicBool::compare_exchange.
