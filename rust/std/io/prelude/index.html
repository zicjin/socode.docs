Module std::io::prelude

Expand description

The I/O Prelude.

The purpose of this module is to alleviate imports of many common I/O traits by adding a glob import to the top of I/O heavy modules:

use std::io::prelude::*;

Re-exports

pub use super::BufRead;
pub use super::Read;
pub use super::Seek;
pub use super::Write;
