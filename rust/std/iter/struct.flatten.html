Struct std::iter::Flatten

pub struct Flatten<I> where    I: Iterator,    <I as Iterator>::Item: IntoIterator,  { /* fields omitted */ }

An iterator that flattens one level of nesting in an iterator of things that can be turned into iterators.

This struct is created by the flatten method on Iterator. See its documentation for more.
