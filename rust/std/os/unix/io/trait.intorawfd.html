Trait std::os::unix::io::IntoRawFd

pub trait IntoRawFd {
    fn into_raw_fd(self) -> RawFd;
}
This is supported on Unix only.
Expand description

A trait to express the ability to consume an object and acquire ownership of its raw file descriptor.

Required methods

fn into_raw_fd(self) -> RawFd

[src]

Consumes this object, returning the raw underlying file descriptor.

This function transfers ownership of the underlying file descriptor to the caller. Callers are then the unique owners of the file descriptor and must close the descriptor once itâ€™s no longer needed.

Example

use std::fs::File;
use std::os::unix::io::{IntoRawFd, RawFd};

let f = File::open("foo.txt")?;
let raw_fd: RawFd = f.into_raw_fd();
