Intl.Segments

An Intl.Segments instance is an iterable collection of the segments of a text string. It is returned by a call to the segment() method of an Intl.Segmenter object.

Try it

Instance methods

Segments.prototype.containing()

Returns an object describing the segment in the original string that includes the code unit at a specified index.

Segments.prototype[@@iterator]()

Returns an iterator to iterate over the segments.
