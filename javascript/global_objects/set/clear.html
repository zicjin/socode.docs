Set.prototype.clear()

The clear() method removes all elements from a Set object.

Try it

Syntax

clear()

Return value

undefined.

Examples

Using the clear() method

const mySet = new Set();
mySet.add(1);
mySet.add('foo');

mySet.size;       // 2
mySet.has('foo'); // true

mySet.clear();

mySet.size;       // 0
mySet.has('bar')  // false
