Division (/)

The division operator (/) produces the quotient of its operands where the left operand is the dividend and the right operand is the divisor.

Syntax

x / y

Examples

Basic division

1 / 2              // 0.5

Math.floor(3 / 2) // 1

1.0 / 2.0         // 0.5

Division by zero

2.0 / 0     // Infinity

2.0 / 0.0   // Infinity, because 0.0 === 0

2.0 / -0.0  // -Infinity
