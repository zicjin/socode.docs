Exponentiation assignment (**=)

The exponentiation assignment operator (**=) raises the value of a variable to the power of the right operand.

Syntax

x **= y // x = x ** y

Examples

Using exponentiation assignment

// Assuming the following variable
//  bar = 5

bar **= 2     // 25
bar **= 'foo' // NaN
