Multiplication (*)

The multiplication operator (*) produces the product of the operands.

Syntax

x * y

Examples

Multiplication using numbers

 2 * 2      // 4
-2 * 2     // -4

Multiplication with Infinity

Infinity * 0         // NaN
Infinity * Infinity  // Infinity

Multiplication with non-numbers

'foo' * 2 // NaN
