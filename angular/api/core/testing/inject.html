inject

function

Allows injecting dependencies in beforeEach() and it().

See more...

inject(tokens: any[], fn: Function): () => any

Parameters
tokens any[]
fn Function
Returns

() => any

Description

Example:

beforeEach(inject([Dependency, AClass], (dep, object) => {
  // some code that uses `dep` and `object`
  // ...
}));

it('...', inject([AClass], (object) => {
  object.doSomething();
  expect(...);
})
