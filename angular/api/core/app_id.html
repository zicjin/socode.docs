APP_ID

const

A DI token representing a unique string ID, used primarily for prefixing application attributes and CSS styles when ViewEncapsulation.Emulated is being used.

See more...

const APP_ID: InjectionToken<string>;

Description

BY default, the value is randomly generated and assigned to the application by Angular. To provide a custom ID value, use a DI provider to configure the root Injector that uses this token.
