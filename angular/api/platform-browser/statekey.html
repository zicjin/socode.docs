StateKey

type-alias

A type-safe key to use with TransferState.

See more...

type StateKey<T> = string & {
    __not_a_string: never;
};

Description

Example:

const COUNTER_KEY = makeStateKey<number>('counter');
let value = 10;

transferState.set(COUNTER_KEY, value);
