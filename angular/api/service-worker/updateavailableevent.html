UpdateAvailableEvent

interface deprecated

An event emitted when a new version of the app is available.

Deprecated: This event is only emitted by the deprecated SwUpdate. Use the VersionReadyEvent instead, which is emitted by SwUpdate. See SwUpdate docs for an example.

interface UpdateAvailableEvent {
  type: 'UPDATE_AVAILABLE'
  current: {...}
  available: {...}
}
