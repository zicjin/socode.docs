NgControlStatus

directive

Directive automatically applied to Angular form controls that sets CSS classes based on control status.

Exported from

  • ReactiveFormsModule
     
  • FormsModule

Selectors

  • [formControlName]
  • [ngModel]
  • [formControl]

Description

CSS classes applied

The following classes are applied as the properties become true:

  • ng-valid
  • ng-invalid
  • ng-pending
  • ng-pristine
  • ng-dirty
  • ng-untouched
  • ng-touched
