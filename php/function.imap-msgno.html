Description

imap_msgno ( resource $imap_stream , int $uid ) : int

Returns the message sequence number for the given uid.

This function is the inverse of imap_uid().

Parameters

imap_stream

An IMAP stream returned by imap_open().

uid

The message UID

Return Values

Returns the message sequence number for the given uid.
