hash_algos

(PHP 5 >= 5.1.2, PHP 7, PHP 8, PECL hash >= 1.1)

hash_algos â€” Return a list of registered hashing algorithms

Description

hash_algos(): array

Parameters

This function has no parameters.

Return Values

Returns a numerically indexed array containing the list of supported hashing algorithms.

Changelog

Version Description
7.4.0 Support for crc32c has been added.
7.1.0 Support for sha512/224, sha512/256, sha3-224, sha3-256, sha3-384 and sha3-512 has been added.

Examples

Example #1 hash_algos() example

As of PHP 7.4.0, hash_algos() will return the following list of algorithm names. 

<?php
print_r(hash_algos());
?>
