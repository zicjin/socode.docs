Description

ldap_rename_ext ( resource $ldap , string $dn , string $new_rdn , string $new_parent , bool $delete_old_rdn [, array|null $controls = null ] ) : resource|false

Does the same thing as ldap_rename() but returns the LDAP result resource to be parsed with ldap_parse_result().

Parameters

See ldap_rename()

Return Values

Returns an LDAP result identifier or false on error.
