public ErrorException::__construct ([ string $message = "" [, int $code = 0 [, int $severity = E_ERROR [, string $filename = __FILE__ [, int $lineno = __LINE__ [, Exception $previous = null ]]]]]] )

Constructs the Exception.

Parameters

message

The Exception message to throw.

code

The Exception code.

severity

The severity level of the exception.

Note:

While the severity can be any int value, it is intended that the error constants be used.

filename

The filename where the exception is thrown.

lineno

The line number where the exception is thrown.

previous

The previous exception used for the exception chaining.
