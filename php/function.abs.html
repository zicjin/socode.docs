abs

(PHP 4, PHP 5, PHP 7)

abs — Absolute value

Description

abs ( mixed $number ) : int|float

Returns the absolute value of number.

Parameters

number

The numeric value to process

Return Values

The absolute value of number. If the argument number is of type float, the return type is also float, otherwise it is int (as float usually has a bigger value range than int).

Examples

Example #1 abs() example

<?php
echo abs(-4.2); // 4.2 (double/float)
echo abs(5);    // 5 (integer)
echo abs(-5);   // 5 (integer)
?>
