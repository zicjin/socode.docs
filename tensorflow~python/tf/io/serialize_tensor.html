tf.io.serialize_tensor

TensorFlow 1 version

Transforms a Tensor into a serialized TensorProto proto.

tf.io.serialize_tensor(
    tensor,
    name=None
)

Args:

  • tensor: A Tensor. A Tensor of type T.
  • name: A name for the operation (optional).

Returns:

A Tensor of type string.
