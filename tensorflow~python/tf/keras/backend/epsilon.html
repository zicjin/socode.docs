tf.keras.backend.epsilon

TensorFlow 1 version View source on GitHub

Returns the value of the fuzz factor used in numeric expressions.

tf.keras.backend.epsilon()

Returns:

A float.

Example:

keras.backend.epsilon() >>>1e-07
