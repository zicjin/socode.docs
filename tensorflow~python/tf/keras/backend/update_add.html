tf.keras.backend.update_add

TensorFlow 1 version View source on GitHub

Update the value of x by adding increment.

tf.keras.backend.update_add(
    x,
    increment
)

Arguments:

  • x: A Variable.
  • increment: A tensor of same shape as x.

Returns:

The variable x updated.
