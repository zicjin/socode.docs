tf.keras.backend.get_uid

TensorFlow 1 version View source on GitHub

Associates a string prefix with an integer counter in a TensorFlow graph.

tf.keras.backend.get_uid(prefix='')

Arguments:

  • prefix: String prefix to index.

Returns:

Unique integer ID.

Example:

get_uid('dense') 
1 
get_uid('dense') 
2
