torch.jit.wait

torch.jit.wait(future) [source]

Forces completion of a torch.jit.Future[T] asynchronous task, returning the result of the task. See fork() for docs and examples. :param func: an asynchronous task reference, created through torch.jit.fork :type func: torch.jit.Future[T]

Returns

the return value of the the completed task
