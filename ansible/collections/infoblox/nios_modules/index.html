Plugin Index

These are the plugins in the infoblox.nios_modules collection:

Modules

  • nios_a_record module – Configure Infoblox NIOS A records
  • nios_aaaa_record module – Configure Infoblox NIOS AAAA records
  • nios_cname_record module – Configure Infoblox NIOS CNAME records
  • nios_dns_view module – Configure Infoblox NIOS DNS views
  • nios_dtc_lbdn module – Configure Infoblox NIOS DTC LBDN
  • nios_dtc_pool module – Configure Infoblox NIOS DTC Pool
  • nios_dtc_server module – Configure Infoblox NIOS DTC Server
  • nios_fixed_address module – Configure Infoblox NIOS DHCP Fixed Address
  • nios_host_record module – Configure Infoblox NIOS host records
  • nios_member module – Configure Infoblox NIOS members
  • nios_mx_record module – Configure Infoblox NIOS MX records
  • nios_naptr_record module – Configure Infoblox NIOS NAPTR records
  • nios_network module – Configure Infoblox NIOS network object
  • nios_network_view module – Configure Infoblox NIOS network views
  • nios_nsgroup module – Configure InfoBlox DNS Nameserver Groups
  • nios_ptr_record module – Configure Infoblox NIOS PTR records
  • nios_restartservices module – Restart grid services.
  • nios_srv_record module – Configure Infoblox NIOS SRV records
  • nios_txt_record module – Configure Infoblox NIOS txt records
  • nios_zone module – Configure Infoblox NIOS DNS zones

Inventory Plugins

  • nios_inventory inventory – Infoblox inventory plugin

Lookup Plugins

  • nios_lookup lookup – Query Infoblox NIOS objects
  • nios_next_ip lookup – Return the next available IP address for a network
  • nios_next_network lookup – Return the next available network range for a network-container
