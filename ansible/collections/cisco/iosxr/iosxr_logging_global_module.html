cisco.iosxr.iosxr_logging_global module – Manages logging attributes of Cisco IOSXR network devices

Note

This module is part of the cisco.iosxr collection (version 3.2.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install cisco.iosxr.

To use it in a playbook, specify: cisco.iosxr.iosxr_logging_global.

New in version 2.4.0: of cisco.iosxr
