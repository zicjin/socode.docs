cisco.nxos.nxos_ospf_vrf module – (deprecated, removed after 2022-10-01)Manages a VRF for an OSPF router.

Note

This module is part of the cisco.nxos collection (version 3.1.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install cisco.nxos.

To use it in a playbook, specify: cisco.nxos.nxos_ospf_vrf.

New in version 1.0.0: of cisco.nxos
