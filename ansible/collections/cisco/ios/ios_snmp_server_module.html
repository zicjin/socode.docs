cisco.ios.ios_snmp_server – snmp_server resource module

Note

This plugin is part of the cisco.ios collection (version 2.6.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install cisco.ios.

To use it in a playbook, specify: cisco.ios.ios_snmp_server.

New in version 2.6.0: of cisco.ios

  • Synopsis
  • Parameters
  • Notes
  • Examples
  • Return Values

Synopsis

  • This module provides declarative management of SNMP server on Cisco IOS devices.

Note

This module has a corresponding action plugin.

              - ftp
