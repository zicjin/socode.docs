Plugin Index

These are the plugins in the cisco.asa collection:

Modules

  • asa_acl module – (deprecated, removed after 2022-06-01) Manage access-lists on a Cisco ASA
  • asa_acls module – Access-Lists resource module
  • asa_command module – Run arbitrary commands on Cisco ASA devices
  • asa_config module – Manage configuration sections on Cisco ASA devices
  • asa_facts module – Collect facts from remote devices running Cisco ASA
  • asa_og module – (deprecated, removed after 2022-06-01) Manage object groups on a Cisco ASA
  • asa_ogs module – Object Group resource module

Cliconf Plugins

  • asa cliconf – Use asa cliconf to run command on Cisco ASA platform
