cisco.aci.aci_tenant_span_src_group_to_dst_group module – Bind SPAN source groups to destination groups (span:SpanLbl)

Note

This module is part of the cisco.aci collection (version 2.2.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install cisco.aci.

To use it in a playbook, specify: cisco.aci.aci_tenant_span_src_group_to_dst_group.

Synopsis

  • Bind SPAN source groups to associated destination groups on Cisco ACI fabrics.

Examples

- cisco.aci.aci_tenant_span_src_group_to_dst_group:
    host: apic
    username: admin
    password: SomeSecretPassword
    tenant: production
    src_group: "{{ src_group }}"
    dst_group: "{{ dst_group }}"
    description: "{{ description }}"
  delegate_to: localhost
