Plugin Index

These are the plugins in the cisco.nso collection:

Modules

  • nso_action module – Executes Cisco NSO actions and verifies output.
  • nso_config module – Manage Cisco NSO configuration and service synchronization.
  • nso_query module – Query data from Cisco NSO.
  • nso_show module – Displays data from Cisco NSO.
  • nso_verify module – Verifies Cisco NSO configuration.
