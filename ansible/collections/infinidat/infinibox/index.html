Description

A set of Ansible modules that allow simple, but powerful, idempotent interactions with Infinidat InfiniBoxes.

Authors:

  • Infinidat <partners.infi@infinidat.com>
  • David Ohlemacher

Plugin Index

These are the plugins in the infinidat.infinibox collection:

Modules

  • infini_cluster module – Create, Delete and Modify Host Cluster on Infinibox
  • infini_export module – Create, Delete or Modify NFS Exports on Infinibox
  • infini_export_client module – Create, Delete or Modify NFS Client(s) for existing exports on Infinibox
  • infini_fs module – Create, Delete or Modify filesystems on Infinibox
  • infini_host module – Create, Delete and Modify Hosts on Infinibox
  • infini_map module – Create and Delete mapping of a volume to a host or cluster on Infinibox
  • infini_pool module – Create, Delete and Modify Pools on Infinibox
  • infini_port module – Add and Delete fiber channel and iSCSI ports to a host on Infinibox
  • infini_user module – Create, Delete and Modify a User on Infinibox
