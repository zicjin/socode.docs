Issue Tracker Repository (Sources)

Plugin Index

These are the plugins in the hetzner.hcloud collection:

Modules

  • hcloud_certificate module – Create and manage certificates on the Hetzner Cloud.
  • hcloud_certificate_info module – Gather infos about your Hetzner Cloud certificates.
  • hcloud_datacenter_info module – Gather info about the Hetzner Cloud datacenters.
  • hcloud_firewall module – Create and manage firewalls on the Hetzner Cloud.
  • hcloud_floating_ip module – Create and manage cloud Floating IPs on the Hetzner Cloud.
  • hcloud_floating_ip_info module – Gather infos about the Hetzner Cloud Floating IPs.
  • hcloud_image_info module – Gather infos about your Hetzner Cloud images.
  • hcloud_load_balancer module – Create and manage cloud Load Balancers on the Hetzner Cloud.
  • hcloud_load_balancer_info module – Gather infos about your Hetzner Cloud Load Balancers.
  • hcloud_load_balancer_network module – Manage the relationship between Hetzner Cloud Networks and Load Balancers
  • hcloud_load_balancer_service module – Create and manage the services of cloud Load Balancers on the Hetzner Cloud.
  • hcloud_load_balancer_target module – Manage Hetzner Cloud Load Balancer targets
  • hcloud_load_balancer_type_info module – Gather infos about the Hetzner Cloud Load Balancer types.
  • hcloud_location_info module – Gather infos about your Hetzner Cloud locations.
  • hcloud_network module – Create and manage cloud Networks on the Hetzner Cloud.
  • hcloud_network_info module – Gather info about your Hetzner Cloud networks.
  • hcloud_placement_group module – Create and manage placement groups on the Hetzner Cloud.
  • hcloud_primary_ip module – Create and manage cloud Primary IPs on the Hetzner Cloud.
  • hcloud_rdns module – Create and manage reverse DNS entries on the Hetzner Cloud.
  • hcloud_route module – Create and delete cloud routes on the Hetzner Cloud.
  • hcloud_server module – Create and manage cloud servers on the Hetzner Cloud.
  • hcloud_server_info module – Gather infos about your Hetzner Cloud servers.
  • hcloud_server_network module – Manage the relationship between Hetzner Cloud Networks and servers
  • hcloud_server_type_info module – Gather infos about the Hetzner Cloud server types.
  • hcloud_ssh_key module – Create and manage ssh keys on the Hetzner Cloud.
  • hcloud_ssh_key_info module – Gather infos about your Hetzner Cloud ssh_keys.
  • hcloud_subnetwork module – Manage cloud subnetworks on the Hetzner Cloud.
  • hcloud_volume module – Create and manage block Volume on the Hetzner Cloud.
  • hcloud_volume_info module – Gather infos about your Hetzner Cloud Volumes.

Inventory Plugins

  • hcloud inventory – Ansible dynamic inventory plugin for the Hetzner Cloud.
