Plugin Index

These are the plugins in the ngine_io.vultr collection:

Modules

  • vultr_account_info module – Get information about the Vultr account.
  • vultr_block_storage module – Manages block storage volumes on Vultr.
  • vultr_block_storage_info module – Get information about the Vultr block storage volumes available.
  • vultr_dns_domain module – Manages DNS domains on Vultr.
  • vultr_dns_domain_info module – Gather information about the Vultr DNS domains available.
  • vultr_dns_record module – Manages DNS records on Vultr.
  • vultr_firewall_group module – Manages firewall groups on Vultr.
  • vultr_firewall_group_info module – Gather information about the Vultr firewall groups available.
  • vultr_firewall_rule module – Manages firewall rules on Vultr.
  • vultr_network module – Manages networks on Vultr.
  • vultr_network_info module – Gather information about the Vultr networks available.
  • vultr_os_info module – Get information about the Vultr OSes available.
  • vultr_plan_baremetal_info module – Gather information about the Vultr Bare Metal plans available.
  • vultr_plan_info module – Gather information about the Vultr plans available.
  • vultr_region_info module – Gather information about the Vultr regions available.
  • vultr_server module – Manages virtual servers on Vultr.
  • vultr_server_baremetal module – Manages baremetal servers on Vultr.
  • vultr_server_info module – Gather information about the Vultr servers available.
  • vultr_ssh_key module – Manages ssh keys on Vultr.
  • vultr_ssh_key_info module – Get information about the Vultr SSH keys available.
  • vultr_startup_script module – Manages startup scripts on Vultr.
  • vultr_startup_script_info module – Gather information about the Vultr startup scripts available.
  • vultr_user module – Manages users on Vultr.
  • vultr_user_info module – Get information about the Vultr user available.

Inventory Plugins

  • vultr inventory – Vultr inventory source
