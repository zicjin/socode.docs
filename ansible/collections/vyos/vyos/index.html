Plugin Index

These are the plugins in the vyos.vyos collection:

Modules

  • vyos_banner module – Manage multiline banners on VyOS devices
  • vyos_bgp_address_family module – BGP Address Family Resource Module.
  • vyos_bgp_global module – BGP Global Resource Module.
  • vyos_command module – Run one or more commands on VyOS devices
  • vyos_config module – Manage VyOS configuration on remote device
  • vyos_facts module – Get facts about vyos devices.
  • vyos_firewall_global module – FIREWALL global resource module
  • vyos_firewall_interfaces module – FIREWALL interfaces resource module
  • vyos_firewall_rules module – FIREWALL rules resource module
  • vyos_hostname module – Manages hostname resource module
  • vyos_interface module – (deprecated, removed after 2022-06-01) Manage Interface on VyOS network devices
  • vyos_interfaces module – Interfaces resource module
  • vyos_l3_interface module – (deprecated, removed after 2022-06-01) Manage L3 interfaces on VyOS network devices
  • vyos_l3_interfaces module – L3 interfaces resource module
  • vyos_lag_interfaces module – LAG interfaces resource module
  • vyos_linkagg module – (deprecated, removed after 2022-06-01) Manage link aggregation groups on VyOS network devices
  • vyos_lldp module – (deprecated, removed after 2022-06-01) Manage LLDP configuration on VyOS network devices
  • vyos_lldp_global module – LLDP global resource module
  • vyos_lldp_interface module – (deprecated, removed after 2022-06-01) Manage LLDP interfaces configuration on VyOS network devices
  • vyos_lldp_interfaces module – LLDP interfaces resource module
  • vyos_logging module – Manage logging on network devices
  • vyos_logging_global module – Logging resource module
  • vyos_ntp_global module – Manages ntp modules of Vyos network devices
  • vyos_ospf_interfaces module – OSPF Interfaces Resource Module.
  • vyos_ospfv2 module – OSPFv2 resource module
  • vyos_ospfv3 module – OSPFV3 resource module
  • vyos_ping module – Tests reachability using ping from VyOS network devices
  • vyos_prefix_lists module – Prefix-Lists resource module for VyOS
  • vyos_route_maps module – Route Map Resource Module.
  • vyos_snmp_server module – Manages snmp_server resource module
  • vyos_static_route module – (deprecated, removed after 2022-06-01) Manage static IP routes on Vyatta VyOS network devices
  • vyos_static_routes module – Static routes resource module
  • vyos_system module – Run set system commands on VyOS devices
  • vyos_user module – Manage the collection of local users on VyOS device
  • vyos_vlan module – Manage VLANs on VyOS network devices

Cliconf Plugins

  • vyos cliconf – Use vyos cliconf to run command on VyOS platform
