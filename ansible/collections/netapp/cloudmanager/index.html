Description

Ansible collection to create CloudManager connectors, CVO instances, CVO aggregates, CVO volumes, and more.

Plugin Index

These are the plugins in the netapp.cloudmanager collection:

Modules

  • na_cloudmanager_aggregate module – NetApp Cloud Manager Aggregate
  • na_cloudmanager_aws_fsx module – Cloud ONTAP file system(FSx) in AWS
  • na_cloudmanager_cifs_server module – NetApp Cloud Manager cifs server
  • na_cloudmanager_connector_aws module – NetApp Cloud Manager connector for AWS
  • na_cloudmanager_connector_azure module – NetApp Cloud Manager connector for Azure.
  • na_cloudmanager_connector_gcp module – NetApp Cloud Manager connector for GCP.
  • na_cloudmanager_cvo_aws module – NetApp Cloud Manager CVO for AWS
  • na_cloudmanager_cvo_azure module – NetApp Cloud Manager CVO/working environment in single or HA mode for Azure.
  • na_cloudmanager_cvo_gcp module – NetApp Cloud Manager CVO for GCP
  • na_cloudmanager_info module – NetApp Cloud Manager info
  • na_cloudmanager_nss_account module – NetApp Cloud Manager nss account
  • na_cloudmanager_snapmirror module – NetApp Cloud Manager SnapMirror
  • na_cloudmanager_volume module – NetApp Cloud Manager volume
