vmware.vmware_rest.vcenter_vm_hardware_adapter_scsi module – Adds a virtual SCSI adapter to the virtual machine.

Note

This module is part of the vmware.vmware_rest collection (version 2.2.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install vmware.vmware_rest.

To use it in a playbook, specify: vmware.vmware_rest.vcenter_vm_hardware_adapter_scsi.

New in version 0.1.0: of vmware.vmware_rest

Synopsis

  • Adds a virtual SCSI adapter to the virtual machine.

Requirements

The below requirements are needed on the host that executes this module.

  • vSphere 7.0.2 or greater
  • python >= 3.6
  • aiohttp

Parameters

Parameter

Comments

adapter

string

Virtual SCSI adapter identifier. Required with state=[‘absent’, ‘present’]

bus

integer

SCSI bus number.

label

string

The name of the item

pci_slot_number

integer

Address of the SCSI adapter on the PCI bus. If the PCI address is invalid, the server will change it when the VM is started or as the device is hot added.

session_timeout

float

added in 2.1.0 of vmware.vmware_rest

Timeout settings for client session.

The maximal number of seconds for the whole operation including connection establishment, request sending and response.

The default value is 300s.

sharing

string

The sharing defines the valid bus sharing modes for a virtual SCSI adapter.

Choices:

  • NONE
  • PHYSICAL
  • VIRTUAL

state

string

Choices:

  • absent
  • present ← (default)

type

string

The type defines the valid emulation types for a virtual SCSI adapter.

Choices:

  • BUSLOGIC
  • LSILOGIC
  • LSILOGICSAS
  • PVSCSI

vcenter_hostname

string / required

The hostname or IP address of the vSphere vCenter

vcenter_password

If the value is not specified in the task, the value of environment variable VMWARE_PASSWORD will be used instead.

Examples

- name: Look up the VM called test_vm1 in the inventory
  register: search_result
  vmware.vmware_rest.vcenter_vm_info:
    filter_names:
    - test_vm1

- name: Collect information about a specific VM
  vmware.vmware_rest.vcenter_vm_info:
    vm: '{{ search_result.value[0].vm }}'
  register: test_vm1_info

- name: Create a SCSI adapter at PCI slot 35
  vmware.vmware_rest.vcenter_vm_hardware_adapter_scsi:
    vm: '{{ test_vm1_info.id }}'
    pci_slot_number: 35

- name: Drop the SCSI controller
  vmware.vmware_rest.vcenter_vm_hardware_adapter_scsi:
    vm: '{{ test_vm1_info.id }}'
    pci_slot_number: 35
    state: absent
