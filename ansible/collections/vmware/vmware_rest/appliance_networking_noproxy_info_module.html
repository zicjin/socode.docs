vmware.vmware_rest.appliance_networking_noproxy_info module – Returns servers for which no proxy configuration will be applied.

Note

This module is part of the vmware.vmware_rest collection (version 2.2.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install vmware.vmware_rest.

To use it in a playbook, specify: vmware.vmware_rest.appliance_networking_noproxy_info.

New in version 2.0.0: of vmware.vmware_rest

Synopsis

  • Returns servers for which no proxy configuration will be applied.

Requirements

The below requirements are needed on the host that executes this module.

  • vSphere 7.0.2 or greater
  • python >= 3.6
  • aiohttp
