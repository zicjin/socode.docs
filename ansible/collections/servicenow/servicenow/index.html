Description

The Service Now modules previously packaged as part of Ansible, now in a Collection distributed via Ansible Galaxy. snow_record creates, updates or deletes a record in a specified table in the Service Now instance. snow_record_find retrieves one or more records based on the supplied query. ServiceNow Inventory Plugin for using ServiceNow CMDB as a dynamic inventory source.
