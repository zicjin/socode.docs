cloud.common.turbo_demo – A demo module for ansible_module.turbo

Note

This plugin is part of the cloud.common collection (version 2.1.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install cloud.common.

To use it in a playbook, specify: cloud.common.turbo_demo.
