This cliconf plugin is part of the dellemc.os6 collection (version 1.0.7).

To use it in a playbook, specify: dellemc.os6.os6.

Synopsis

  • This os6 plugin provides low level abstraction apis for sending and receiving CLI commands from Dell OS6 network devices.
