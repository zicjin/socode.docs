Plugin Index

These are the plugins in the dellemc.os10 collection:

Modules

  • base_xml_to_dict module – Operations for show command output convertion from xml to json format.
  • bgp_validate module – Validate the bgp neighbor state,raise error if it is not in established state
  • mtu_validate module – Validate the MTU value for lldp neighbors
  • os10_command module – Run commands on devices running Dell EMC SmartFabric OS10
  • os10_config module – Manage Dell EMC SmartFabric OS10 configuration sections
  • os10_facts module – Collect facts from devices running Dell EMC SmartFabric OS10
  • show_system_network_summary module – Operations for show_system_network output in json/yaml format.
  • vlt_validate module – Validate the vlt info, raise an error if peer is not in up state
  • wiring_validate module – Validate the wiring based on the planned wiring details

Cliconf Plugins

  • os10 cliconf – Use os10 cliconf to run command on Dell OS10 platform
