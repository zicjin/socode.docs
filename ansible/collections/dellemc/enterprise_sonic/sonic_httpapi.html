dellemc.enterprise_sonic.sonic httpapi – HttpApi Plugin for devices supporting Restconf SONIC API

Note

This httpapi plugin is part of the dellemc.enterprise_sonic collection (version 1.1.1).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install dellemc.enterprise_sonic.

To use it in a playbook, specify: dellemc.enterprise_sonic.sonic.

New in version 1.0.0: of dellemc.enterprise_sonic

  • Synopsis
  • Parameters

Synopsis

  • This HttpApi plugin provides methods to connect to Restconf SONIC API endpoints.

Parameters

Parameter

Comments

root_path

string

Specifies the location of the Restconf root.

Default: “/restconf”

Configuration:

  • Variable: ansible_httpapi_restconf_root

Hint

Configuration entries for each entry type have a low to high priority order. For example, a variable that is lower in the list will override a variable that is higher up.
