Plugin Index

These are the plugins in the community.grafana collection:

Modules

  • grafana_dashboard module – Manage Grafana dashboards
  • grafana_datasource module – Manage Grafana datasources
  • grafana_folder module – Manage Grafana Folders
  • grafana_notification_channel module – Manage Grafana Notification Channels
  • grafana_organization module – Manage Grafana Organization
  • grafana_plugin module – Manage Grafana plugins via grafana-cli
  • grafana_team module – Manage Grafana Teams
  • grafana_user module – Manage Grafana User

Callback Plugins

  • grafana_annotations callback – send ansible events as annotations on charts to grafana over http api.

Lookup Plugins

  • grafana_dashboard lookup – list or search grafana dashboards
