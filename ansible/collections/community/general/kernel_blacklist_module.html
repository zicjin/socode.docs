community.general.kernel_blacklist module – Blacklist kernel modules

Note

This module is part of the community.general collection (version 5.4.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install community.general.

To use it in a playbook, specify: community.general.kernel_blacklist.

  • Synopsis
  • Parameters
  • Examples

Synopsis

  • Add or remove kernel modules from blacklist.

Parameters

Parameter

Comments

blacklist_file

string

If specified, use this blacklist file instead of /etc/modprobe.d/blacklist-ansible.conf.

Default: “/etc/modprobe.d/blacklist-ansible.conf”

name

string / required

Name of kernel module to black- or whitelist.

state

string

Whether the module should be present in the blacklist or absent.

Choices:

  • absent
  • present ← (default)

Examples

- name: Blacklist the nouveau driver module
  community.general.kernel_blacklist:
    name: nouveau
    state: present
