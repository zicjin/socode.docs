community.general.logentries_msg module – Send a message to logentries.

Note

This module is part of the community.general collection (version 5.4.0).

You might already have this collection installed if you are using the ansible package. It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install community.general.

To use it in a playbook, specify: community.general.logentries_msg.

Synopsis

  • Send a message to logentries

Requirements

The below requirements are needed on the host that executes this module.

  • python >= 2.6

Parameters

Parameter

Comments

api

string

API endpoint

Default: “data.logentries.com”

msg

string / required

The message body.

port

integer

API endpoint port

Default: 80

token

string / required

Log token.

Examples

- name: Send a message to logentries
  community.general.logentries_msg:
    token=00000000-0000-0000-0000-000000000000
    msg="{{ ansible_hostname }}"
