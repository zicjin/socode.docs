Plugin Index

These are the plugins in the community.proxysql collection:

Modules

  • proxysql_backend_servers module – Adds or removes mysql hosts from proxysql admin interface
  • proxysql_global_variables module – Gets or sets the proxysql global variables
  • proxysql_info module – Gathers information about proxysql server
  • proxysql_manage_config module – Writes the proxysql configuration settings between layers
  • proxysql_mysql_users module – Adds or removes mysql users from proxysql admin interface
  • proxysql_query_rules module – Modifies query rules using the proxysql admin interface
  • proxysql_query_rules_fast_routing module – Modifies query rules for fast routing policies using the proxysql admin interface
  • proxysql_replication_hostgroups module – Manages replication hostgroups using the proxysql admin interface
  • proxysql_scheduler module – Adds or removes schedules from proxysql admin interface
