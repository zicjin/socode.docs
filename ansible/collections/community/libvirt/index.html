Plugin Index

These are the plugins in the community.libvirt collection:

Modules

  • virt module – Manages virtual machines supported by libvirt
  • virt_net module – Manage libvirt network configuration
  • virt_pool module – Manage libvirt storage pools

Connection Plugins

  • libvirt_lxc connection – Run tasks in lxc containers via libvirt
  • libvirt_qemu connection – Run tasks on libvirt/qemu virtual machines

Inventory Plugins

  • libvirt inventory – Libvirt inventory source
