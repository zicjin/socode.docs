Plugin Index

These are the plugins in the community.mysql collection:

Modules

  • mysql_db module – Add or remove MySQL databases from a remote host
  • mysql_info module – Gather information about MySQL servers
  • mysql_query module – Run MySQL queries
  • mysql_replication module – Manage MySQL replication
  • mysql_role module – Adds, removes, or updates a MySQL role
  • mysql_user module – Adds or removes a user from a MySQL database
  • mysql_variables module – Manage MySQL global variables
