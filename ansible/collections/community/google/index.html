Plugin Index

These are the plugins in the community.google collection:

Modules

  • gc_storage module – This module manages objects/buckets in Google Cloud Storage.
  • gce_eip module – Create or Destroy Global or Regional External IP addresses.
  • gce_img module – utilize GCE image resources
  • gce_instance_template module – create or destroy instance templates of Compute Engine of GCP.
  • gce_labels module – Create, Update or Destroy GCE Labels.
  • gce_lb module – create/destroy GCE load-balancer resources
  • gce_mig module – Create, Update or Destroy a Managed Instance Group (MIG).
  • gce_net module – create/destroy GCE networks and firewall rules
  • gce_pd module – utilize GCE persistent disk resources
  • gce_snapshot module – Create or destroy snapshots for GCE storage volumes
  • gce_tag module – add or remove tag(s) to/from GCE instances
  • gcpubsub module – Create and Delete Topics/Subscriptions, Publish and pull messages on PubSub
  • gcpubsub_info module – List Topics/Subscriptions and Messages from Google PubSub.

Lookup Plugins

  • gcp_storage_file lookup – Return GC Storage content
