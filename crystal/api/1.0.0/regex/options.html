Enum Members

IGNORE_CASE = 1

Case insensitive match.

MULTILINE = 6

PCRE native PCRE_MULTILINE flag is 2, and PCRE_DOTALL is 4

  • PCRE_DOTALL changes the "." meaning
  • PCRE_MULTILINE changes "^" and "$" meanings

Crystal modifies this meaning to have essentially one unique "m" flag that activates both behaviours, so here we do the same by mapping MULTILINE to PCRE_MULTILINE | PCRE_DOTALL.

EXTENDED = 8

Ignore white space and # comments.

ANCHORED = 16

Force pattern anchoring.
