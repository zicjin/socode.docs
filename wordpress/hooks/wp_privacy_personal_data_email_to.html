apply_filters( 'wp_privacy_personal_data_email_to', string $request_email, WP_User_Request $request )

Filters the recipient of the personal data export email notification.

Description

Should be used with great caution to avoid sending the data export link to wrong emails.

Parameters

$request_email

(string) The email address of the notification recipient.

$request

(WP_User_Request) The request that is initiating the notification.
