apply_filters( 'plugin_auto_update_setting_html', string $html, string $plugin_file, array $plugin_data )

Filters the HTML of the auto-updates setting for each plugin in the Plugins list table.

Parameters

$html

(string) The HTML of the plugin's auto-update column content, including toggle auto-update action links and time to next update.

$plugin_file

(string) Path to the plugin file relative to the plugins directory.

$plugin_data

(array) An array of plugin data. See get_plugin_data() and the 'plugin_row_meta' filter for the list of possible values.
