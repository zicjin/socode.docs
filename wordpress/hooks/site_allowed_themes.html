apply_filters( 'site_allowed_themes', string[] $allowed_themes, int $blog_id )

Filters the array of themes allowed on the site.

Parameters

$allowed_themes

(string[]) An array of theme stylesheet names.

$blog_id

(int) ID of the site. Defaults to current site.
