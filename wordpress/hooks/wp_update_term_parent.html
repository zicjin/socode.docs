apply_filters( 'wp_update_term_parent', int $parent, int $term_id, string $taxonomy, array $parsed_args, array $args )

Filters the term parent.

Description

Hook to this filter to see if it will cause a hierarchy loop.

Parameters

$parent

(int) ID of the parent term.

$term_id

(int) Term ID.

$taxonomy

(string) Taxonomy slug.

$parsed_args

(array) An array of potentially altered update arguments for the given term.

$args

(array) An array of update arguments for the given term.
