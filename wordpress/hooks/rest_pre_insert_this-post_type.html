apply_filters( "rest_pre_insert_{$this->post_type}", stdClass $prepared_post, WP_REST_Request $request )

Filters a post before it is inserted via the REST API.

Description

The dynamic portion of the hook name, $this->post_type, refers to the post type slug.

Possible hook names include:

  • rest_pre_insert_post
  • rest_pre_insert_page
  • rest_pre_insert_attachment

Parameters

$prepared_post

(stdClass) An object representing a single post prepared for inserting or updating the database.

$request

(WP_REST_Request) Request object.
