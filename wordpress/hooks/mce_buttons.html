apply_filters( 'mce_buttons', array $mce_buttons, string $editor_id )

Filters the first-row list of TinyMCE buttons (Visual tab).

Parameters

$mce_buttons

(array) First-row list of buttons.

$editor_id

(string) Unique editor identifier, e.g. 'content'. Accepts 'classic-block' when called from block editor's Classic block.
