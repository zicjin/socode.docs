do_action( 'saved_term', int $term_id, int $tt_id, string $taxonomy, bool $update )

Fires after a term has been saved, and the term cache has been cleared.

Description

The ‘saved_$taxonomy’ hook is also available for targeting a specific taxonomy.

Parameters

$term_id

(int) Term ID.

$tt_id

(int) Term taxonomy ID.

$taxonomy

(string) Taxonomy slug.

$update

(bool) Whether this is an existing term being updated.
