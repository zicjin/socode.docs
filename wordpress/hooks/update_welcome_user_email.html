apply_filters( 'update_welcome_user_email', string $welcome_email, int $user_id, string $password, array $meta )

Filters the content of the welcome email after user activation.

Description

Content should be formatted for transmission via wp_mail().

Parameters

$welcome_email

(string) The message body of the account activation success email.

$user_id

(int) User ID.

$password

(string) User password.

$meta

(array) Signup meta data. Default empty array.
