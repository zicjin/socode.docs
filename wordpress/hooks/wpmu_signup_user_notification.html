apply_filters( 'wpmu_signup_user_notification', string $user_login, string $user_email, string $key, array $meta )

Filters whether to bypass the email notification for new user sign-up.

Parameters

$user_login

(string) User login name.

$user_email

(string) User email address.

$key

(string) Activation key created in wpmu_signup_user().

$meta

(array) Signup meta data. Default empty array.
