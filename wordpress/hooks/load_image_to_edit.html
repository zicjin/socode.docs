apply_filters( 'load_image_to_edit', resource|GdImage $image, int $attachment_id, string|int[] $size )

Filters the current image being loaded for editing.

Parameters

$image

(resource|GdImage) Current image.

$attachment_id

(int) Attachment ID.

$size

(string|int[]) Requested image size. Can be any registered image size name, or an array of width and height values in pixels (in that order).
