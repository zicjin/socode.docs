apply_filters( "{$field}", mixed $value, int $post_id, string $context )

Filters the value of a specific post field for display.

Description

The dynamic portion of the hook name, $field, refers to the post field name.

Parameters

$value

(mixed) Value of the prefixed post field.

$post_id

(int) Post ID.

$context

(string) Context for how to sanitize the field. Accepts 'raw', 'edit', 'db', 'display', 'attribute', or 'js'. Default 'display'.
