apply_filters( 'load_image_to_edit_attachmenturl', string|false $image_url, int $attachment_id, string|int[] $size )

Filters the path to an attachment’s URL when editing the image.

Description

The filter is only evaluated if the file isn’t stored locally and allow_url_fopen is enabled on the server.

Parameters

$image_url

(string|false) Current image URL.

$attachment_id

(int) Attachment ID.

$size

(string|int[]) Requested image size. Can be any registered image size name, or an array of width and height values in pixels (in that order).
