apply_filters( 'rest_prepare_comment', WP_REST_Response $response, WP_Comment $comment, WP_REST_Request $request )

Filters a comment returned from the REST API.

Description

Allows modification of the comment right before it is returned.

Parameters

$response

(WP_REST_Response) The response object.

$comment

(WP_Comment) The original comment object.

$request

(WP_REST_Request) Request used to generate the response.
