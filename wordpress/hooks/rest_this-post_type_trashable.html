apply_filters( "rest_{$this->post_type}_trashable", bool $supports_trash, WP_Post $post )

Filters whether a post is trashable.

Description

The dynamic portion of the hook name, $this->post_type, refers to the post type slug.

Possible hook names include:

  • rest_post_trashable
  • rest_page_trashable
  • rest_attachment_trashable

Pass false to disable Trash support for the post.

Parameters

$supports_trash

(bool) Whether the post type support trashing.

$post

(WP_Post) The Post object being considered for trashing support.
