apply_filters( 'network_home_url', string $url, string $path, string|null $orig_scheme )

Filters the network home URL.

Parameters

$url

(string) The complete network home URL including scheme and path.

$path

(string) Path relative to the network home URL. Blank string if no path is specified.

$orig_scheme

(string|null) Scheme to give the URL context. Accepts 'http', 'https', 'relative' or null.
