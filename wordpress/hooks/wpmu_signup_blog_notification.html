apply_filters( 'wpmu_signup_blog_notification', string|false $domain, string $path, string $title, string $user_login, string $user_email, string $key, array $meta )

Filters whether to bypass the new site email notification.

Parameters

$domain

(string|false) Site domain, or false to prevent the email from sending.

$path

(string) Site path.

$title

(string) Site title.

$user_login

(string) User login name.

$user_email

(string) User email address.

$key

(string) Activation key created in wpmu_signup_blog().

$meta

(array) Signup meta data. By default, contains the requested privacy setting and lang_id.
