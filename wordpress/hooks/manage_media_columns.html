apply_filters( 'manage_media_columns', string[] $posts_columns, bool $detached )

Filters the Media list table columns.

Parameters

$posts_columns

(string[]) An array of columns displayed in the Media list table.

$detached

(bool) Whether the list table contains media not attached to any posts. Default true.
