do_action( 'rest_after_insert_attachment', WP_Post $attachment, WP_REST_Request $request, bool $creating )

Fires after a single attachment is completely created or updated via the REST API.

Parameters

$attachment

(WP_Post) Inserted or updated attachment object.

$request

(WP_REST_Request) Request object.

$creating

(bool) True when creating an attachment, false when updating.
