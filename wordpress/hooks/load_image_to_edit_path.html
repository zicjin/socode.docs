apply_filters( 'load_image_to_edit_path', string|false $filepath, int $attachment_id, string|int[] $size )

Filters the returned path or URL of the current image.

Parameters

$filepath

(string|false) File path or URL to current image, or false.

$attachment_id

(int) Attachment ID.

$size

(string|int[]) Requested image size. Can be any registered image size name, or an array of width and height values in pixels (in that order).
