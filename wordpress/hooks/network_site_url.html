apply_filters( 'network_site_url', string $url, string $path, string|null $scheme )

Filters the network site URL.

Parameters

$url

(string) The complete network site URL including scheme and path.

$path

(string) Path relative to the network site URL. Blank string if no path is specified.

$scheme

(string|null) Scheme to give the URL context. Accepts 'http', 'https', 'relative' or null.
