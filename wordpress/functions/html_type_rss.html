html_type_rss()

Display the HTML type based on the blog setting.

Description

The two possible values are either ‘xhtml’ or ‘html’.

Source

File: wp-includes/feed.php 

function html_type_rss() {
	$type = get_bloginfo( 'html_type' );
	if ( strpos( $type, 'xhtml' ) !== false ) {
		$type = 'xhtml';
	} else {
		$type = 'html';
	}
	echo $type;
}
