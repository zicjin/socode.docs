wp_cache_set_sites_last_changed()

Sets the last changed time for the ‘sites’ cache group.

Source

File: wp-includes/ms-site.php 

function wp_cache_set_sites_last_changed() {
	wp_cache_set( 'last_changed', microtime(), 'sites' );
}
