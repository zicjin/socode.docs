comment_date( string $format = '', int|WP_Comment $comment_ID )

Displays the comment date of the current comment.

Parameters

$format

(string) (Optional) PHP date format. Defaults to the 'date_format' option.

Default value: ''

$comment_ID

(int|WP_Comment) (Optional) WP_Comment or ID of the comment for which to print the date. Default current comment.

More Information

For the format of the date, see Formatting Date and Time.

Source

File: wp-includes/comment-template.php 

function comment_date( $format = '', $comment_ID = 0 ) {
	echo get_comment_date( $format, $comment_ID );
}
