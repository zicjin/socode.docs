wp_ajax_time_format()

Ajax handler for time formatting.

Source

File: wp-admin/includes/ajax-actions.php 

function wp_ajax_time_format() {
	wp_die( date_i18n( sanitize_option( 'time_format', wp_unslash( $_POST['date'] ) ) ) );
}
