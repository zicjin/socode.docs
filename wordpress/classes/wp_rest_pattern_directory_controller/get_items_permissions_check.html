WP_REST_Pattern_Directory_Controller::get_items_permissions_check( WP_REST_Request $request )

Checks whether a given request has permission to view the local block pattern directory.

Parameters

$request

(WP_REST_Request) (Required) Full details about the request.

Return

(true|WP_Error) True if the request has permission, WP_Error object otherwise.

Source

File: wp-includes/rest-api/endpoints/class-wp-rest-pattern-directory-controller.php 

public function get_items_permissions_check( $request ) {
		if ( current_user_can( 'edit_posts' ) ) {
			return true;
		}

		foreach ( get_post_types( array( 'show_in_rest' => true ), 'objects' ) as $post_type ) {
			if ( current_user_can( $post_type->cap->edit_posts ) ) {
				return true;
			}
		}

		return new WP_Error(
			'rest_pattern_directory_cannot_view',
			__( 'Sorry, you are not allowed to browse the local block pattern directory.' ),
			array( 'status' => rest_authorization_required_code() )
		);
	}
