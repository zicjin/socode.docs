WP_REST_Widgets_Controller::get_item_permissions_check( WP_REST_Request $request )

Checks if a given request has access to get a widget.

Parameters

$request

(WP_REST_Request) (Required) Full details about the request.

Return

(true|WP_Error) True if the request has read access, WP_Error object otherwise.

Source

File: wp-includes/rest-api/endpoints/class-wp-rest-widgets-controller.php 

public function get_item_permissions_check( $request ) {
		$this->retrieve_widgets();

		$widget_id  = $request['id'];
		$sidebar_id = wp_find_widgets_sidebar( $widget_id );

		if ( $sidebar_id && $this->check_read_sidebar_permission( $sidebar_id ) ) {
			return true;
		}

		return $this->permissions_check( $request );
	}
