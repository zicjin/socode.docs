wpdb::db_version()

Retrieves the database server version.

Return

(string|null) Version number on success, null on failure.

Source

File: wp-includes/wp-db.php 

public function db_version() {
		return preg_replace( '/[^0-9.].*/', '', $this->db_server_info() );
	}
