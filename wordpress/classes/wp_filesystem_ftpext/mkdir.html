WP_Filesystem_FTPext::mkdir( string $path, int|false $chmod = false, string|int|false $chown = false, string|int|false $chgrp = false )

Creates a directory.

Parameters

$path

(string) (Required) Path for new directory.

$chmod

(int|false) (Optional) The permissions as octal number (or false to skip chmod).

Default value: false

$chown

(string|int|false) (Optional) A user name or number (or false to skip chown).

Default value: false

$chgrp

(string|int|false) (Optional) A group name or number (or false to skip chgrp).

Default value: false

Return

(bool) True on success, false on failure.

public function mkdir( $path, $chmod = false, $chown = false, $chgrp = false ) {
		$path = untrailingslashit( $path );

		if ( empty( $path ) ) {
			return false;
		}

		if ( ! ftp_mkdir( $this->link, $path ) ) {
			return false;
		}

		$this->chmod( $path, $chmod );

		return true;
	}
