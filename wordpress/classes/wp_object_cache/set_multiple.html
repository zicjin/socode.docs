WP_Object_Cache::set_multiple( array $data, string $group = '', int $expire )

Sets multiple values to the cache in one call.

Parameters

$data

(array) (Required) Array of key and value to be set.

$group

(string) (Optional) Where the cache contents are grouped.

Default value: ''

$expire

(int) (Optional) When to expire the cache contents, in seconds. Default 0 (no expiration).

Return

(bool[]) Array of return values, grouped by key. Each value is always true.

Source

File: wp-includes/class-wp-object-cache.php 

public function set_multiple( array $data, $group = '', $expire = 0 ) {
		$values = array();

		foreach ( $data as $key => $value ) {
			$values[ $key ] = $this->set( $key, $value, $group, $expire );
		}

		return $values;
	}
