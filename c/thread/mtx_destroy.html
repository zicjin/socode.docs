mtx_destroy

Defined in header <threads.h> 
void mtx_destroy( mtx_t *mutex );
 (since C11)

Destroys the mutex pointed to by mutex.

If there are threads waiting on mutex, the behavior is undefined.

Parameters

mutex - pointer to the mutex to destroy

Return value

(none).
